# e-plantShopping

Welcome to **e-plantShopping**, a modern e-commerce web application designed for plant enthusiasts to browse, discover, and purchase their favorite plants online. 

## Project Overview
This application features an intuitive shopping cart experience where users can seamlessly add plants to their cart, adjust quantities, and see their real-time totals before checkout. It is built using a fast, component-driven frontend architecture to ensure a smooth user experience.

## Features
* **Plant Catalog:** Browse a diverse collection of indoor and outdoor plants.
* **Interactive Shopping Cart:** Dynamic updating of items, quantities, and total costs.
* **Responsive Design:** Fully optimized for both desktop and mobile screens.
